The Python Data Model

Presented by: Josh Karpel

Thursday, August 8 2019, 6:00 PM
Madison Public Library, 201 W Mifflin St, Room 301 [Map]
RSVP on Meetup (Note: This Meetup has already occurred.)

Learn about Python's "data model", which lets you write beautiful, idiomatic Python by making your objects behave like native parts of the language. We'll also touch on Domain-Driven Design and API design.
